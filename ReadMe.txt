Model Setup:
I. Code environment:
	1. create virtual environment based on env.txt file in the project root dir

II. Download data for model*:
	1. Download URL to Traffic Signs Images (4GB in size): https://www.kaggle.com/datasets/valentynsichkar/traffic-signs-preprocessed/download
	2. Once downloaded, unzip archive.zip and copy data0.pickle (455 MB) to ./data
	* A slice of the data0.pickle (268 MB) is packaged with this submission and can be used for testing the script

III. Launch Jupyter Notebook: final_project_road_sign_cnn.ipynb
